Arrow Functions

Modern & Elegant Function Syntax

What are Arrow Functions?

A compact and elegant way to write functions in JavaScript. They provide a shorter syntax and lexical this binding.

Syntax Examples

Traditional Function 
function sum(a, b) {
    return a + b;
}
Arrow Function 
const sum = (a, b) => {
    return a + b;
}

Even Shorter Syntax

One-liner 
const sum = (a, b) => a + b;

Shorter Syntax

Less code, same functionality

Lexical This

No more binding issues

Implicit Return

Return without keyword
